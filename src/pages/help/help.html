Once you have recorded your data, a list of the records will be displayed here. You can also switch to see the records displayed on a map view using the tab buttons above. Click on a record in the list to continue editing that record.

Uploading Records

When one or more records have been completed, you can upload them to the server by clicking the upload button. Completed records (shown in green) are those where all required fields are entered and within the allowable constraints. You can still upload the completed records even if there are incomplte records (shown in blue), although the latter will not be uploaded.

Upload Data

Tree Sightings

When working on a census, you can create one or more tree sightings. Select the Tree Sightings List tab at the top of the page and use the button below to create a new tree sighting.

Create a Tree Sighting

GPS accuracy

The accuracy of the GPS in your device will improve over time after it has been started. You may wish to update the location information in your observation or census periodically if the current accuracy is insufficient. This can be done by pressing the "{{UPDATE_BUTTON_NAME}}" button within a record form.
